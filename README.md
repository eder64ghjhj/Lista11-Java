# Lista11-Java

Lista de exercícios em Java com foco em **estruturas de repetição (`for`)**.

## 📋 Exercícios

1. Soma dos números de 1 a 100
2. Exibição dos números pares entre 0 e 50
3. Produto dos números de 1 a 10 (fatorial)
4. Tabuada de um número informado pelo usuário
5. Fatorial de um número informado pelo usuário
6. Sequência de Fibonacci
7. Contagem de números divisíveis por 3 entre 1 e 50
8. Quadrado dos números de 1 a 20
9. Média de 10 números informados pelo usuário
10. Contagem de números primos entre 1 e 100

## 🛠️ Tecnologias

- Java

## ▶️ Como executar

```bash
javac ExercN.java
java lista11.ExercN
```
